- Pandas
- Matplotlib
- Scikit-learn

## Algorithm
- Random Forest Classifier

## Features

- Species distribution visualization
- Model training
- Accuracy evaluation
- Classification report
- Feature importance graph
- Interactive flower prediction

## Output
- Accuracy
- Confusion Matrix
- Classification Report
- Saved ML Model
